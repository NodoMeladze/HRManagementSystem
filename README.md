# HRManagementSystem

# სატესტო პროექტი.

# პროექტის გამოყენების ინსტრუქცია:
1) დაკლონეთ რეპოზიტორი.
2) API-ის appsettings.json და appsettings.Development.json ფაილებში ჩაასწორეთ ქონექშენ სთრინგი, თქვენთვის სასურველით.
3) API-ს გაშვებისას ავტომატურად მოხდება ბაზის ინიციალიზაცია და ბეისიქ დეითას სიდინგი (ფენდინგ მიგრაცია უკვე არის პროექტში).
4) გაუშვით MVC აპლიკაცია
5) დაარეგისტრირეთ იუზერი და ავტომატურად დალოგინდებით სისტემაში.
